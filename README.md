# Grandmother's Europe and Asia Photo Gallery
A beautiful photo gallery showcasing travel memories across Europe and Asia.
Overview..
This project is a curated collection of photographs documenting my grandmother's adventures and experiences traveling through Europe and Asia. The gallery provides an organized way to view, browse, and share these cherished memories.
Features

Easy Navigation - Browse photos by region (Europe and Asia)
Responsive Design - Works seamlessly on desktop, tablet, and mobile devices
High-Quality Images - Displays photos in their best light
Simple Organization - Photos are organized for easy browsing
Lightweight - Fast loading times for smooth viewing

Getting Started
Prerequisites

A modern web browser (Chrome, Firefox, Safari, Edge)
No additional installations required

Installation

Clone this repository:

bash   git clone https://github.com/yourusername/grandmother-photo-gallery.git
   cd grandmother-photo-gallery

Open index.html in your web browser, or serve locally:

bash   python -m http.server 8000

Visit http://localhost:8000 in your browser
